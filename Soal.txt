Buatlah sebuah game tebak angka sederhana.
Program akan melakukan random angka 1-100
pengguna akan menebak angka yang 
sudah dirandom oleh program.

Untuk menerima tebakan, buatlah agar program bisa menerima
input dari user.
Selama tebakan tidak tepat, program akan terus berjalan.
Jika tebakan benar, maka tampilkan 
"Selamat anda berhasil menebak angka <angka_yg_ditebak>"
Jika ingin exit, silahkan inputkan 999 / "exit" (pilih salah
satu).
